- ## キーボードとタイピングについて
 
- あなたが常用しているキーボードの製品名を教えてください（例 Realforce, HHK, Majestouch）。またそれを使用する理由も教えてください
- あなたが使用するキーボードの種類はどれですか？ 1.JIS / 2.US / 3.その他。また、それを選んだ理由も教えてください
- あなたが使用するキー配列はどれですか？ 1.QWERTY / 2.Dvorak / 3.その他。また、それを選んだ理由も教えてください
- あなたのタイピング力を教えていただけませんか（タイピングゲーム等のスコアだとわかりやすいです）
- タッチタイピングはできますか？できる場合、どれならできますか？ 1.文字キー / 2.数字キー / 3.記号 / 4.修飾やファンクションなど特殊キー
- OSやアプリのショートカットキーはどのようにして使っていますか？ 1.デフォルトのまま使う / 2.変更可能な場合は適宜変更して使う / 3. リマップツール等で無理矢理変更して使う
- あなたが最も多く押しているキー、ベスト 3 を挙げてください
- 「ここは譲れない」というIME（日本語入力ソフトウェア）の設定やこだわりがありましたら教えてください（例 スペースは常に半角入力, 学習機能は常にオフ, ●●な単語は辞書登録しておく）
